# Product Notes

## Included in this rebuild

- Professional React application shell
- Asset registry and asset-level tabs
- Browser-side profiling for CSV and `.xlsx`
- Rule-based starter DQ evaluation
- Record-level Overall Quality
- Cross-asset quality and observability dashboard
- Schema mismatch confirmation
- Safe checkbox-based run deletion
- Run comparison charts
- Issue workflow
- Cloudflare static deployment configuration

## Deliberately not presented as complete yet

- Multi-user authentication and RBAC
- Server-side scheduled monitoring
- Email, Slack, or Teams notifications
- Durable shared storage
- Governed custom rule builder
- Database and cloud-storage connectors
- Audit logs and approval workflows

These require the shared Cloudflare backend in `ARCHITECTURE.md` and should not be simulated as working controls.
